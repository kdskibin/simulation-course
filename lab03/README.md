### Клеточные автоматы. Лесные пожары (GUI)

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.


**Решение:**
Для моделирования возникновения и распространения лесных пожаров с использования двумерного клеточного автомата был исопльзован язык программирования python и библиотеки: matplotlib, numpy.

Программа предоставляет возможность выбрать размер области моделирования (количество клеток), начальную заполненность деревьями, направление ветра и его сила, коффициент влияния температуры, а также вероятности: возгорание дерева, возникновения дерева на пустой клетке, возгорания дерева по диагонали от текущей клетки.

**Дополнительные правила системы:**
- Направление ветра
- Сила ветра
- Коэффициент влияния температуры
- Вероятность возгорания дерева по диагонали
- Искусственный водоём в центре области моделирования

**Пример работы системы:**

В данном примере используются следующие парметры:
Ветер: юго-западный
Влияние ветра: 0.8
Вероятность появления нового дерева: 0.8
Вероятность возникновения огня: 0.001
Коэффициент влияния температуры: 1.1
Размер моделируемой области: 400*400

**Выводы:**

По итогу выполнения лабораторной работы была реализована модель возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата. Данная модель позволяет учитывать направление ветра и его силу, что может быть полезно для моделирования реальных реальных пожаров и поиска способов по их предотвращению. Верно, что данная модель далека от реального мира, поскольку не учитывается рельеф местности, влажность воздуха, возраст леса, время года и многие другие факторы, оказывающие значительное влияние на распространение огня.
